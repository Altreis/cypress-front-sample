# BDD base project: Cypress + Cucumber + Typescript

Basic example of using Cypress with Cucumber (BDD). 


# Get started

## Installation

```bash
npm install
```

# Running tests manually

Open **`Cypress`** and run the tests manually:

```bash
npm run cypress:open
```
